# Advanced Data Visualization Practicals

## Overview

This repository contains a collection of academic data visualization practicals completed during my M.Tech coursework.

The objective of these practicals was to explore a wide variety of visualization techniques, chart types, and analytical dashboards for presenting complex datasets in an understandable and interactive manner.
